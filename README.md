## 🌟 Introduction

**LM Certificate Publisher Pro** is a world-class, extremely versatile digital credentialing and secure verification platform. It is engineered from the ground up to issue, design, manage, and verify digital certificates, legal documents, security licenses, and professional compliance records at any scale.

This is a **highly adaptable, industrial-strength system** built to power the verification needs of high-profile private corporations, legal institutes, security firms, online education academies, and official agencies worldwide.

With LM Certificate Publisher Pro, you can eliminate forgery, automate credential distribution, and provide instant, beautiful verification portals. Everything is managed directly in your WordPress dashboard, giving you total ownership of your valuable data.

## 📡 Secure REST API & Key Generation (Ready to Use)

### 🔑 Secure Random Key Generator (12-16 Digits)
The plugin includes a powerful **Secure Random Key Generator** custom field type. 
*   **Instant Keys:** When publishing a new document, license, or credential, the plugin can automatically generate a highly secure, random **12 to 16-character alphanumeric key** (e.g., `X97F-K2P4-99LS-QW12`).
*   **Use Cases:** Perfect for software licenses, physical security badges, tamper-proof credentials, and high-value legal documents. Users can verify their status simply by pasting this unique key into your search forms.

### 🔌 Easy REST API Setup
We believe that enterprise data should be highly accessible yet incredibly secure. LM Certificate Publisher Pro comes equipped with a built-in, easy-to-use **REST API**.
*   **Third-Party Verification:** Easily verify document status, serial keys, validity, and custom fields from external websites, mobile apps, or third-party systems safely.

## 📦 Installation

### Method 1: Upload via WordPress Admin
1. Download the plugin ZIP archive containing the `/lm-certificate-publisher-pro/` folder.
2. Log in to your WordPress Dashboard.
3. Navigate to **Plugins → Add New** and click **Upload Plugin** at the top.
4. Select the ZIP file and click **Install Now**.
5. Once uploaded, click **Activate Plugin**.
6. The gold-accented **🏆 LM Certificates** menu will immediately appear in your sidebar!

### Method 2: FTP Installation
1. Extract the plugin ZIP file on your computer.
2. Upload the `/lm-certificate-publisher-pro/` folder to your server's `/wp-content/plugins/` directory.
3. Go to **Plugins** in WordPress and activate **LM Certificate Publisher Pro**.

## ⚡ Quick Start Guide

### Step 1: Claim Your License Key
1. Go to **LM Certificates → License** in your dashboard.
2. Enter your commercial license key (or secure 60-day trial key).
3. Click **Activate License** to unlock premium features and live updates.

### Step 2: Create Custom Fields
1. Navigate to **LM Certificates → Custom Fields**.
2. Add any parameters your documents require (e.g., `officer_id`, `audit_score`, `expiry_date`).
3. These fields instantly display on your document editing page.

### Step 3: Design with Elementor
1. Create a new Elementor Template (Section recommended).
2. Drag and drop our **Certificate Publisher** widgets.
3. Bind widgets to your custom fields using Elementor's native **Dynamic Tags**.

### Step 4: Add WooCommerce Selling (Optional Premium Add-on)
1. Install and activate our premium paid **LM WooCommerce Extension**.
2. Add a new product in WooCommerce and set its type to "Certificate Product".
3. Configure plans, pricing tiers, and fields.
4. Watch the beautiful, step-by-step multi-step checkout wizard activate automatically on your shop page!

### Step 5: Publish & Verify
1. Go to **LM Certificates → Add New Certificate**.
2. Input the recipient's details, select your design template, and click **Publish**.
3. Place the shortcode `[lmc_certificate_search]` on any page to launch your instant public verification portal!

## 🖥️ Admin Dashboard

### 1. Document & Certificate Management
The dashboard provides a smooth, elegant table view of all issued credentials. Easily search, bulk-edit, filter by category or tag, and copy verification URLs with one click.
Every document automatically receives:
- A unique **8-character verification code** for quick online checkups.
- An incremental, custom-prefixed **Serial Number** (e.g., `SEC-2026-00104`).
- A sharp, vector **QR code** for mobile validation.

### 2. Custom Fields
Create as many dynamic fields as you need with zero limits. Reorder them using a smooth drag-and-drop grid. 
Supported field types include:
-   **Text & Numbers:** Perfect for grades, IDs, rankings, and names.
-   **URLs & File Uploads:** Ideal for downloadable PDFs, audio logs, spreadsheets, and project files.
-   **Tables:** Built-in table editor to present complex data grids, subjects, or compliance checklists beautifully.

### 3. Categories & Tags
Organize your digital documents using standard WordPress hierarchies. Perfect for categorizing records by department, year, training track, or security level.

### 4. Search Settings
Fully customize your public search experience. Define exactly which database columns (Roll number, Name, License Key, Email, Phone, etc.) are queryable. Add category filters, loading screens, and search bar text labels easily.

### 5. Advanced Settings
Take control of advanced performance, styling, and security features:
-   **Styling Customization:** Inject your own custom CSS code directly to format borders and containers.
-   **Google reCAPTCHA v3:** Keep database queries lightning fast and protected from malicious scrapers.
-   **Watermark Seal:** Overlay custom translucent images/logos over your certificates to block screenshots and forgery.
-   **Preloader Animations:** Choose from beautiful CSS spinners or upload your own branded GIF.
-   **Expiry Rules:** Control whether expired licenses should display custom warning banners or disappear entirely from public views.
-   **Automation Emails:** Customize professional HTML email notifications sent instantly when credentials are published.
-   **Whitelabeling Settings:** Fully replace plugin credits with your own branding. Upload a custom admin menu logo, customize plugin titles, hide "Powered by" footprints, and configure tailored styling options for a 100% personalized client hand-off.
-   **Advanced Copy Protection:** Easily toggle right-click (context-menu) deactivation and disable text/image selection on certificate views to block unauthorized downloads, credential scraping, and visual forgery.

### 6. License Management
Track your active licensing tiers, view site allocations, and deactivate sites securely to release licensing slots for development or client handovers.

## 📄 Landscape PDF Export Engine

Say goodbye to distorted or blurry PDFs. Our engine utilizes high-quality client-side rendering with `html2pdf.js`:
-   **Desktop Fidelity:** Clones and preserves high-resolution fonts, logos, and layouts perfectly.
-   **Safety Margin Buffer:** Adds a custom `+40px` height margin buffer to guarantee signatures and decorative borders are never clipped at page breaks.
-   **Mobile Fallback:** Intelligently redirects mobile devices to server-side rendering for optimal speed and reliability.

## 🛡️ Security Features

We prioritize data integrity and database speed:
-   **Anti-Theft Copy Protection:** Prevents direct certificate scraping and theft by completely disabling right-click context menus, image dragging, and text selection highlights on frontend certificate display pages.
-   **reCAPTCHA v3 Protection:** Defends public search forms from bot attacks and brute-force inquiries.
